# API & Backend Validation Project

## Role: Software Tester / QA Engineer

This repository documents API testing and backend validation activities performed on a REST-based backend system supporting a web application.

## Responsibilities Covered
- Performed API testing for request and response validation
- Validated HTTP status codes, payload structure, and error handling
- Designed test data for edge cases and negative scenarios
- Ensured data consistency between frontend and backend systems

## API Testing Scope
- Authentication APIs
- User management APIs
- Negative and edge case validation
- Backend data consistency checks

## Tools & Technologies
- Python
- Pytest
- Requests library
- REST APIs
- JSON
- Git & GitHub

> API endpoints and business details are intentionally anonymized to maintain confidentiality.
